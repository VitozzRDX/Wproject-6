# Wproject-6 — Интерактивная гексагональная карта

Браузерный просмотрщик тактической карты с гексагональной сеткой. Карта разделена на гексы, поверх растровых изображений отображаются слои рельефа: здания, лес, холмы, дороги. Навигация — клавишами WASD.

## Структура проекта

```
index.html          — точка входа, подключает Konva и main.js
main.js             — инициализация и сборка всех слоёв
map.js              — загрузка карты (3 GIF-файла, склеенные вертикально)
hexgrid.js          — отрисовка гексагональной сетки поверх карты
buildings.js        — слой зданий (загружается из buildings_cache.png)
forest.js           — слой леса (загружается из forest_cache.png)
hills.js            — слой холмов (загружается из hills_cache.png)
roads.js            — слой дорог (векторная отрисовка)
brush.js            — инструмент выделения/рисования по гексам
controls.js         — управление навигацией (WASD)
konva.min.js        — библиотека Konva.js (2D canvas)
```

### Растровые ресурсы

| Файл | Описание |
|------|----------|
| `1.gif`, `2.gif`, `3.gif` | Три части карты, склеиваются вертикально |
| `bdp.gif` | Вспомогательное изображение карты |
| `buildings_cache.png` | Предрендеренный слой зданий |
| `forest_cache.png` | Предрендеренный слой леса |
| `hills_cache.png` | Предрендеренный слой холмов |

## Архитектура слоёв

Карта строится из нескольких Konva-слоёв, которые накладываются друг на друга:

```
[map]        — базовые изображения карты (1.gif / 2.gif / 3.gif)
[hexgrid]    — гексагональная сетка
[roads]      — дороги
[buildings]  — здания
[forest]     — лес
[hills]      — холмы
[brush]      — инструмент выделения
```

## Кэширование слоёв

Слои зданий, леса и холмов вычисляются один раз путём анализа цветов пикселей исходной карты и сохраняются как PNG-файлы (`*_cache.png`). При загрузке страницы они просто отображаются как изображения — без пересчёта. Это значительно ускоряет запуск.

## Гексагональная сетка

- Гексы с плоским верхом, радиус `R = 112/3 px`
- Колонки чередуются со смещением по вертикали (offset grid)
- Адресация гексов: буквенно-цифровая (A1, B2, ..., AA1 и т.д.) и специальный формат для нулевой строки

## Управление

| Клавиша | Действие |
|---------|----------|
| `W` | Прокрутка вниз |
| `S` | Прокрутка вверх |
| `A` | Прокрутка вправо |
| `D` | Прокрутка влево |

## Запуск

Требуется локальный HTTP-сервер (файлы не открываются напрямую через `file://` из-за ES-модулей и загрузки изображений).

```bash
# Например, через Python
python -m http.server 8080

# Или через Node.js (npx serve)
npx serve .
```

Затем открыть `http://localhost:8080` в браузере.

## Технологии

- [Konva.js](https://konvajs.org/) — 2D canvas рендеринг
- Vanilla JS (ES modules) — без фреймворков и сборщиков
